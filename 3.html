Measures of Central Tendency

Measures of central tendency se hum data ka center samajhte hain,
- matlab wo value jo data ka typical ya representative value hoti hai.
- Yeh measures batate hain ke data kahan clustered hai.

  • Core Measures of Central Tendency:

    • Mean (Arithmetic Average)
    • Median
    • Mode
  • 1 Mean
    • Mean ko hum average bhi kehte hain.
    • Yeh wo value hoti hai jo data set ke central point ko represent karti hai.
    • Iska calculation simple hai—sabhi numbers ko add karo aur unka total number of items se divide karo.

Feature Mean Median Mode
Kya batata hai? Average value Middle value Most frequent value
Outlier sensitive? ✅ Yes ❌ No ❌ No
Categorical pe kaam? ❌ Nahi ❌ Limited ✅ Perfect
Numerical data? ✅ Yes ✅ Yes ✅ Yes
Condition / Situation ✅ Use Mean ✅ Use Median ✅ Use Mode 🔎 Explanation
📈 Balanced numerical data ✅ Yes ✅ Yes ❌ Rarely Mean aur Median dono accurate center dikhate hain
⚠️ Data mein outliers hon ❌ No ✅ Yes ❌ Maybe Median zyada reliable hoti hai
🔀 Skewed (uneven) data ❌ No ✅ Yes ❌ No Median real center show karta hai
📉 Normal (bell-shaped) distribution ✅ Yes ✅ Yes ✅ Maybe Sab close hotay hain — koi bhi kaam kar sakta hai
🔡 Categorical data (e.g. Shirt, Pant) ❌ No ❌ No ✅ Yes Mode hi kaam karta hai — most frequent value
🧩 Ordinal data (e.g. Poor, Good, Excellent) ❌ No ✅ Yes ✅ Yes Median + Mode dono useful — average rank ya most frequent
🧼 Missing value fill — numerical balanced ✅ Yes ✅ Yes ❌ No Mean ya Median use hoti hain
🧼 Missing value fill — skewed/outlier data ❌ No ✅ Yes ❌ No Median safe hoti hai from distortion
🧼 Missing value fill — categorical ❌ No ❌ No ✅ Yes Mode sabse zyada common value fill karta hai
🧮 Data center chahiye (center of tendency) ✅ Yes ✅ Yes ❌ No Center ke liye mean/median better hotay hain
📊 Data summary (quick idea) ✅ Yes ✅ Yes ✅ Yes Teeno ka use ek sath bhi hota hai
🤖 ML: Standardization (x - mean / std) ✅ Required ❌ No ❌ No Mean required hota hai Standard Scaler ke liye
🤖 ML: Robust Scaler / Skewed Data ❌ No ✅ Yes ❌ No Median based scaling zyada stable hoti hai
🧠 User behavior ya product trend analysis ❌ No ❌ No ✅ Yes Mode se most common item find hota hai

Feature 🔍 🟠 Simple (Arithmetic) Mean 🟢 Geometric Mean
📐 Formula (x₁ + x₂ + x₃ + ... + xₙ) / n (x₁ × x₂ × x₃ × ... × xₙ)^(1/n)
➕ Data kis type ka hota hai? Jab values add ho rahi ho (e.g. marks) Jab values multiply ho rahi ho (e.g. growth rate)
🔁 Calculation logic Sab values ko add karo aur divide karo Sab values ko multiply karo aur root lo
📈 Outlier ka effect Bohat zyada hota hai Kam hota hai
➗ Zero allow karta hai? ✅ Haan ❌ Nahi (0 multiply hone se result 0 ban jata hai)
➖ Negative values? ✅ Haan ❌ Nahi (odd roots of negative = complex number)
🧠 Use kab hota hai? Jab values ki simple average chahiye Jab values ratios ya % growth represent kar rahi ho
🎯 Real-world example Marks, height, age, exam score, salary Investment growth, interest rate, population growth
🧮 Result interpretation Central value of data Realistic average of %age or ratios
📊 Graph mein look Center of gravity Slightly pulled toward smaller values
🎯 Use karo jab... ➕ Addition-based situations (score, marks) ✖️ Multiplication-based (growth, interest, CPI)
Situation Use This Mean Reason
Student ke marks ka average ✅ Arithmetic Mean Marks add hote hain
Paisa 3 saal tak grow ho +10%, +30% ✅ Geometric Mean Compound growth multiply hota hai
Height ka average ✅ Arithmetic Mean Normal addition-based metric
Price Index (CPI, inflation) ✅ Geometric Mean Prices multiply hoti hain (inflation effect)
Machine learning ke ratios / growth ✅ Geometric Mean Ratios aur scale zyada accurate hota hai
Mixed + Negative values present ✅ Arithmetic Mean Geometric mean invalid hoti hai negative/0 pe

🔢 Mean Type 🧠 Core Difference 🧮 Simple Equation 🎯 Kab Use Karein? (Use-case)
🟠 Arithmetic Mean Sab values ka additive average (x1 + x2 + x3 + ... + xn) ÷ n Jab values add ho rahi ho (marks, salary, height)
🟢 Geometric Mean Values ka multiplicative average nth root of (x1 × x2 × x3 × ... × xn) Jab values percent ya ratio form mein ho (growth, ROI)
🟣 Harmonic Mean Values ka inverse average (per-unit based) n ÷ (1/x1 + 1/x2 + ... + 1/xn) Jab values speed ya rate ho (distance/speed, Rs/unit)
Type Use When...
Arithmetic Mean Jab normal average chahiye (additive context)
Geometric Mean Jab growth, percentage ya compound effect ho
Harmonic Mean Jab rate, speed, ya unit-per-time cheezen ho
